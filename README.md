# nputop-ts

TypeScript version of nputop - Intel NPU monitoring tool. This is a TypeScript port of the original [nputop](https://github.com/DMontgomery40/nputop) Python tool.

## Installation

```bash
npm install
npm run build
```

## Usage

```bash
npm start
```

## Development

```bash
npm run build -- --watch
```

## Docker

```bash
docker-compose up --build
```
